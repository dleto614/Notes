#### Vimrc

Just my vim config file. I ripped this from somewhere online and it works for what I want since I didn't want or need a whole lot of plugins.

I use VSCodium for bigger programming projects, but this works well for small scripts and programs. Also use to edit other files.

-----

```bash
set nocompatible
syntax on
set modelines=0
set number
set encoding=utf-8
set wrap

set tabstop=2
set shiftwidth=2
set softtabstop=2
set autoindent
set copyindent
set expandtab
set noshiftround

set hlsearch
set incsearch
set showmatch
set smartcase

set hidden
set ttyfast
set laststatus=2

set showcmd
set background=dark

set shortmess=atI
```
